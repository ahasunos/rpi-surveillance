This file should be on the Raspberry Pi which has the setup of the surveillance system with the motion sensor and camera. 

To execute the program: 
1. Place the file in any location of your raspberry Pi and change the path accordingly in the program.
2. Insert the token of your dropbox in the appropriate area. 
3. Change the GPIO pin accordingly where the motion sensor is inserted.
4. We're good to go, once we checked the above things. Run the program.
